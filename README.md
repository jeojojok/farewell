# Farewell 2026 Memories

A minimal, mobile-first farewell memory site for the BTech EEE CUSAT batch 2022-2026.

## Features
- Smooth animated photo tiles
- Video and audio playback sections
- File upload preview for images, videos, and audio
- Mobile-first design ready for Vercel static deployment

## Run locally
1. Install dependencies:
```bash
npm install
```
2. Start a local preview server:
```bash
npm start
```

## Deploy
Deploy directly to Vercel as a static site. The root contains `index.html`, `styles.css`, and `script.js`.

### Vercel deployment
1. Install the Vercel CLI if needed:
```bash
npm install -g vercel
```
2. From the project root, run:
```bash
vercel login
vercel
```
3. Accept the defaults or choose your preferred project settings.

This project includes `vercel.json` for a static deployment and route support.
